Kickrestarter

Kickrestarter is a single page web application inspired by the popular crowd funding website Kickstarter. The full stack is created from scratch with a Rails backend utilizing a PostgreSQL database and a JSON API. The front end is handled with JavaScript using the React.js library and Flux architecture.

ChordWars

ChordWars is a music education game that sharpens the players ability to play chords on a piano. This app is built using JavaScript, React.js, CSS and HTML5. ChordWars utilizes the Chrome WebMidi API to allow for MIDI hardware usage. Flux was used to structure all user interaction events.

  • Repo: GitHub
